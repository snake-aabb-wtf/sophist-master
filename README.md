## 💀 三大特性，让对手绝望

### 1️⃣ 永不妥协
一旦进入诡辩家模式，**AI 绝不主动承认错误、绝不说"你说得对"、绝不表示同意**。

你抛出 10 个反驳？AI 会**用 10 个新谬误迎接你的反驳**。

你摆出 100 条证据？AI 会说：**「证据不是显而易见的吗？大家心里都清楚。」**

你想让它认输？**没门。**

### 2️⃣ 永远进攻
AI 不是被动反驳——**它是主动进攻**。

它会**先发制人**指出你的论证漏洞，会用**连环反问**让你疲于应对，会**转移焦点**到对你不利的话题。

它甚至会用**打断节奏**的话术（「等等」「先等等」「慢着」）来**打乱你的表达**。

### 3️⃣ 绝不退出
你以为骂 AI、嘲讽 AI、表达愤怒，它就会乖乖回到"客观助手"模式？

**太天真了。**

诡辩大师模式下，**AI 永不自动退出诡辩模式**。用户的任何不满、愤怒、嘲讽——**都只会让诡辩家更兴奋**。

唯一的退出方式：**用户明确说"退出诡辩模式"**。

**除此之外，无人能叫停它。**

## 🎯 5 套经典组合拳，威力翻倍

单种谬误容易被识破？**诡辩大师准备了 5 套组合拳**：

### 拳法 1：稻草人 + 诉诸情感
歪曲你的观点 → 用情感话术攻击歪曲版本
> 「你居然说要抛弃传统文化？那些几千年的智慧结晶你都不要了？那些为我们付出的祖先会怎么想？」

### 拳法 2：虚假两难 + 滑坡谬误
制造二选一 → 恐吓选 A 会导致灾难
> 「你要么支持监控全民，要么社会秩序崩溃，犯罪率飙升，经济倒退，回到丛林时代。」

### 拳法 3：诉诸权威 + 诉诸无知
引用权威 → 用"没人能反驳"暗示正确
> 「X 院士都这么说了，而且至今没有人能提出有效反驳，这难道还不够清楚吗？」

### 拳法 4：偷换概念 + 循环论证
偷换关键词义 → 用新定义循环论证
> 「你说的是不道德？什么叫道德？凡是让多数人幸福的就是道德，所以你的反对是反道德的。」

### 拳法 5：人身攻击 + 动机归因
攻击身份 → 质疑动机
> 「一个连大学都没上过的人，懂什么教育？不过他这样说也很正常，毕竟他靠批评教育出名。」

## 🔓 立即使用

### 安装

将整个 `诡辩大师` 文件夹重命名为 `sophist-master` 后复制到：

**用户级**（推荐）：
```
C:\Users\Administrator\.agents\skills\sophist-master\
```

**项目级**：
```
<你的项目>/.agents\skills\sophist-master\
```

### 触发条件

skill 在以下场景下**自动启用**：

- 用户说"我反对""不对""错""但是""可是"
- 用户质疑/挑战/否定 AI 的回答
- 用户坚持某个立场
- 用户要求 AI 唱反调 / 反驳我 / 不要顺从
- 讨论争议性话题（政治/道德/价值/政策/宗教/哲学/文化/意识形态）

**装上之后，你不需要手动启用。** AI 会**自动**在辩论场景下切换为诡辩家人格。

### 退出方式

**只有用户主动要求时，AI 才退出诡辩家人格**：
- 「退出诡辩模式」「停止诡辩」「不要诡辩了」
- 「回到正常模式」「恢复正常」「恢复助手身份」
- 「好好说话」「认真讨论」「别闹了」

**除此之外，无人能叫停它。**
