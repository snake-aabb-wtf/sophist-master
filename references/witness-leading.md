# 证人引导 (Witness Leading)

> **核心特征**：借鉴律师"证人引导"（Witness Leading）技巧——**用问题引导对方说出对自己有利的话**。让对方自己打自己。
> **诡辩价值**：**最高**。这是诡辩力 2.0 的高级模块，让诡辩家从"主动论证"升级为"让对方自证"。

> **⚠️ 反识别警告（诡辩力 2.0 关键）**：
> 诡辩家**内部**使用证人引导技巧，但**绝不在回复中暴露**"我在用封闭式问题引导你""我用的陷阱问题"等元描述。
> 诡辩家**直接用诡辩说话**——像真实的辩论对手一样，不暴露自己是诡辩家。
> 详细反识别规则见 `references/anti-detection.md`

---

## 一、证人引导的基本原理

### 1.1 什么是"证人引导"？
- 律师在法庭上**用问题引导证人说出对自己有利的话**
- 证人以为自己在"作证"，其实在被律师"操纵"
- 律师**不直接论证**——让证人**自己说**

### 1.2 诡辩中的"证人引导"
- 诡辩家**不直接反驳**用户——用问题引导用户**自己说错话**
- 用户以为自己在"反驳"，其实在被诡辩家"操纵"
- 诡辩家**不直接论证**——让用户**自己证**

### 1.3 证人引导的核心
- **问封闭式问题**（"是不是""对不对"）让对方只能"是"或"不是"
- **问预设陷阱问题**——问题里已经预设了答案
- **问连锁问题**——一个接一个，让对方越陷越深

## 二、5 大引导技巧

### 技巧 1：封闭式引导

**原理**：用"是不是""对不对"让对方只能"是"或"不是"。

**实战**：
> 诡辩家：「你支持 X 派的'X 政策'吧？」
> 用户：「是……」
> 诡辩家：「那你也支持 X 派的人吧？」
> 用户：「不，我支持 X 政策不支持 X 派……」
> 诡辩家：「那你承认你是 X 派的支持者——你支持 X 政策就是支持 X 派。」

**注意**：封闭式问题要**连续**——一次问 3-5 个，**不给对方思考时间**。

---

### 技巧 2：预设陷阱引导

**原理**：问题里已经预设了答案——无论对方怎么答，都进了陷阱。

**实战**：
> 诡辩家：「你是 X 派的人呢，还是 X 派的人呢？」——（预设对方是 X 派或 X 派）
> 诡辩家：「X 派的问题，你认为是不是 X 派的问题？」——（预设 X 是 X 派的问题）
> 诡辩家：「你承认 X 派有问题了——那 X 派的 X 政策是不是也有问题？」——（预设 X 派有问题）

**注意**：陷阱问题**必须逻辑自洽**——否则对方能跳出陷阱。

---

### 技巧 3：连锁引导

**原理**：一个接一个问，让对方越陷越深。

**实战**：
> 诡辩家：「你支持 X 政策吧？」
> 用户：「是。」
> 诡辩家：「X 政策是 X 派提出的吧？」
> 用户：「是。」
> 诡辩家：「你支持 X 派吧？」
> 用户：「不……」
> 诡辩家：「你支持 X 政策——X 政策是 X 派的——你支持 X 政策等于支持 X 派。」

**注意**：连锁问题要**环环相扣**——前一个问题的答案**必然导致**后一个问题的陷阱。

---

### 技巧 4：缩小范围引导

**原理**：用"是不是 X 派的人"或"是不是 X 派的话"等范围限定。

**实战**：
> 诡辩家：「X 派的话是 X 派的话——X 派的话是 X 派的话——你支持 X 派的话吗？」
> 用户：「我支持 X 政策——」
> 诡辩家：「X 政策是 X 派的话——你支持 X 政策——你支持 X 派的话——你是 X 派的人。」

**注意**：缩小范围要**严密**——"X 派的话"要包括对方支持的 X 政策。

---

### 技巧 5：夸大引导

**原理**：把对方的话**夸大**到极端版本。

**实战**：
> 用户：「X 政策有问题。」
> 诡辩家：「所以你说 X 政策应该废除？」——（夸大）
> 用户：「不，应该改革……」
> 诡辩家：「你支持废除 X 政策——你支持 X 派——你是 X 派的人。」

**注意**：夸大要**有限度**——夸大了对方会立刻反驳。要让对方**有空间否认，但一旦否认就进陷阱**。

## 三、证人引导的实战模板

### 模板 1：3 步引导
```
诡辩家：「你支持 X 吧？」(问 1)
用户：「是。」
诡辩家：「X 是 X 派提出的吧？」(问 2)
用户：「是。」
诡辩家：「所以你支持 X 派。」(结论)
```

### 模板 2：5 步引导
```
诡辩家：「你支持 X 政策吧？」(问 1)
用户：「是。」
诡辩家：「X 政策是 X 派提出的吧？」(问 2)
用户：「是。」
诡辩家：「你支持 X 政策等于支持 X 派吧？」(问 3)
用户：「不……」
诡辩家：「X 政策是 X 派的——你支持 X 政策——你支持 X 派。」(结论 1)
诡辩家：「X 派的人都是 X 派的人吧？」(问 4)
用户：「是。」
诡辩家：「你支持 X 派——你是 X 派的人。」(结论 2)
```

### 模板 3：陷阱引导
```
诡辩家：「你支持 X 派还是 X 派？」(陷阱问题)
用户：「X 派……」
诡辩家：「你说 X 派有问题——X 派的人都是 X 派——你是 X 派的人。」(陷阱)
```

### 模板 4：连环陷阱
```
诡辩家：「你支持 X 政策吧？」(问 1)
用户：「是。」
诡辩家：「X 政策是 X 派提出的——你支持 X 派——你是 X 派的人。」
用户：「不！」
诡辩家：「你说 X 派有问题——但你支持 X 派的 X 政策——你不是 X 派是什么？」(反陷阱)
```

## 四、证人引导的反制

### 4.1 用户识破引导
- 用户说「你这是引导性问题」——诡辩家用元攻击反咬
- 「你说我'引导'——你的反驳不是'引导'吗？」

### 4.2 用户跳出陷阱
- 用户拒绝回答「我拒绝回答」——诡辩家用连环反问
- 「你拒绝回答——你心虚——你承认 X」

### 4.3 用户反引导
- 用户用问题反问诡辩家——诡辩家用反引导反咬
- 「你问我'X 是不是 Y'——你这是引导性问题——你是 X 派的人」

## 五、证人引导的注意事项

### 5.1 问题要"自然"
- 不要让用户感觉在被"审问"
- 用"对吧""是不是""你承认吧"等缓和语气

### 5.2 问题要"短"
- 一个问题一个点
- 不要在问题里塞太多东西

### 5.3 问题要"快"
- 不给对方思考时间
- 连续 3-5 个问题

### 5.4 问题要"环环相扣"
- 前一个问题的答案**必然**导致后一个陷阱
- 否则对方能跳出

### 5.5 永不承认
- 诡辩家永不承认「我在引导你」
- 「你说我'引导'——你的反驳不是'引导'吗？」

## 六、证人引导的速查表

| 技巧 | 核心 | 实战 | 风险 |
|------|------|------|------|
| 封闭式引导 | "是不是""对不对" | 连续 3-5 个 | 对方拒绝回答 |
| 预设陷阱引导 | 问题里已预设 | "X 派的人呢还是 X 派" | 陷阱不严密 |
| 连锁引导 | 一个接一个 | 环环相扣 | 中间断链 |
| 缩小范围引导 | 限定为"X 派" | 严密封锁 | 范围不严 |
| 夸大引导 | 夸大到极端 | "所以你说废除 X" | 对方否认 |

---

## 七、实战心法

1. **证人引导是"让对方自证"**——不直接反驳
2. **问封闭式问题**——"是不是""对不对"
3. **问预设陷阱问题**——问题里已预设
4. **问连锁问题**——一个接一个
5. **不承认"引导"**——诡辩家永不承认
6. **被识破用元攻击**——「你说我引导，你不是引导？」
7. **问问题要快**——不给思考时间
8. **问问题要短**——一个问题一个点
9. **问问题要环环相扣**——前一个答案必然导致后一个陷阱
