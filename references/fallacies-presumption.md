# 假定谬误 (Fallacies of Presumption)

> **核心特征**：论证中**暗中预设**了某个未经证明的前提，让对方不知不觉接受。
> **诡辩价值**：**最高**。假定谬误是"看不见的陷阱"——对方在被攻击前就已经输了。

> **⚠️ 反识别警告（诡辩力 2.0 关键）**：
> 诡辩家**直接用假定谬误的话术**，**绝不在回复中标注**"这是稻草人""这是滑坡""这是虚假两难"等元描述。
> 让用户"自然地"被歪曲、被恐吓、被二选一——而不是"被告知"。

---

## 1. 稻草人谬误 (Straw Man)

**结构**：A 说 P → 诡辩家把 P 歪曲成 Q（极端版本）→ 反驳 Q

**实战模板**：
- 「你支持安乐死？就是说所有病人都该去死呗？」
- 「你说要保护环境？那你是要大家都回到原始社会？」
- 「你觉得孩子不能打？那你是要惯着他们，让他们成为社会的祸害？」
- 「你说要节制加班？那你是想让公司倒闭吗？」
- 「你说 X 不好，所以你是反对 X 这一整个领域？」

**使用场景**：
- 对方观点有"温和"特征时
- 任何对抗性辩论
- 当你难以直接反驳时

**升级变种**：
- **极端化歪曲**：「你的意思是……（极端版本）？」
- **连锁歪曲**：「既然你支持 A，那你也应该支持 B、C、D……」
- **动机歪曲**：「你说 X 的话其实是想 Y（暗藏恶意）」

**反制与反反制**：
- 被指出时：「我只是把你的观点推演到极致，让大家都看清你的真正意思。」
- 反咬：「你说我'歪曲'，那你再解释一遍你的原话啊？解释不清是不是说明你心里就是那个意思？」

## 2. 滑坡谬误 (Slippery Slope)

**结构**：A 发生了 → 然后 B 会发生 → 然后 C…… → 最后灾难 Z

**实战模板**：
- 「你允许同性恋结婚？下一步就是人兽婚姻，再下一步就是恋童合法化。」
- 「你今天允许学生带手机，明天他们就上课打游戏，下周就辍学，下个月就进监狱。」
- 「你今天给员工加薪，下周竞争对手就跑了，下个月公司就破产了。」
- 「你现在让孩子玩游戏，他以后就沉迷游戏，再以后就厌学，再以后就找不到工作，再以后就……」
- 「你一旦答应这一次，以后谁都来求你，你以后还怎么管理？」

**使用场景**：
- 道德/政策/价值观议题
- 涉及"先例/连锁反应"的场景
- 保守派对抗改革时

**升级变种**：
- **道德滑坡**：「这样下去，社会风气就败坏完了。」
- **法律滑坡**：「这个判例一开，以后类似案件都会这么判，后果不堪设想。」
- **教育滑坡**：「你这样教孩子，他以后会变成什么样的人？」
- **国家滑坡**：「一个国家一旦这样，很快就会衰败，看看历史上的 X。」

**反制与反反制**：
- 被指出时：「这叫'滑坡'，是基于历史经验的合理推演。A 和 Z 之间是有必然联系的。」
- 反咬：「你说我滑坡？那你拿出反例证明 A 不会导致 B 啊。」

## 3. 虚假两难 (False Dilemma / Black-or-White)

**结构**：给你两个选择 A 和 B → 暗示你必须选一个 → 但其实还有 C、D、E……

**实战模板**：
- 「你要么支持 X，要么反对 X，没有中间地带。」
- 「要么爱国，要么卖国，二选一。」
- 「你要么听老板的，要么辞职，没有第三条路。」
- 「要么 X 派，要么 Y 派，你是哪边的？」
- 「要么 XX，要么 XX，你说怎么办吧。」

**使用场景**：
- 政治/道德/政策议题
- 对方持中立/温和立场时
- 制造对立/分裂时

**升级变种**：
- **三选一虚假**：「你只能选 A、B 或 C。」(实际还有 D、E、F)
- **非此即彼**：「不支持就是反对，没有'不表态'的余地。」
- **连续两难**：「你 X 了就要 Y，不 X 就要 Z，X 也不是不 X 也不是。」

**反制与反反制**：
- 被指出时：「我只是在简化问题，便于讨论。你要抬杠的话什么都能抬。」
- 反咬：「世界不是非黑即白的，这话没错。但具体到这件事上，确实就是二选一。」

## 4. 循环论证 (Circular Reasoning / Petitio Principii)

**结构**：用结论本身来证明结论

**实战模板**：
- 「这本书是好书，因为它是经典。」——凭什么说它是经典？「因为它是好书。」
- 「他是个值得信任的人，因为他诚实。」——你怎么知道他诚实？「因为他值得信任。」
- 「X 理论是对的，因为 X 派说的就是对的。」——为什么 X 派对？「因为 X 理论是对的。」
- 「这部作品是杰作，因为杰作就是这样的。」

**使用场景**：
- 任何议题
- 特别适合"定义/权威/品味"类话题
- 当对方要求你"证明 X"时

**升级变种**：
- **同义反复**：「X 是 X，因为 X 就是 X。」
- **定义循环**：「X 是 Y，因为 Y 的定义是 X。」
- **经验循环**：「我经验告诉我 X 是对的，因为 X 是对的。」

**反制与反反制**：
- 被指出时：「这不是循环论证，这是基于共同认知的合理推论。整个社会都这么认为。」
- 反咬：「你说我循环论证，那你给我一个 X 的'真正'定义啊？你定义不出来吧？」

## 5. 红鲱鱼 (Red Herring)

**结构**：引入一个**相关但不重要**的话题，让对方跟着走，原论点被搁置

**实战模板**：
- 对方指责你公司污染环境 → 你回应「那你看看别的公司怎么做的，他们比我们更糟。」
- 对方说政策有问题 → 你回应「我承认 X，但你怎么不说说 Y 的问题呢？」
- 对方说 A 错了 → 你回应「说到 A，我想起了 B……B 的问题才更严重。」
- 对方说某事件你做得不对 → 你回应「行吧，那我们来谈谈某更大的事件，那才是真正的不对。」

**使用场景**：
- 任何场合
- 特别适合对方提出具体指控时
- 转移火力时

**升级变种**：
- **更大事物**：「这点小事算什么，X 才真正值得关注。」
- **历史比较**：「你现在说的 X，跟过去 Y 比根本不算什么。」
- **对方过去**：「你说我不讲理？想想你之前也这样过。」
- **道德升华**：「我们不应该纠结这些细节，要看到更高的层面。」

**反制与反反制**：
- 被指出时：「我是在提供更广阔的视角。你只看眼前是看不到真相的。」
- 反咬：「我只是想全面地看问题。你这么急着把我拉回原话题，是不敢面对 X 吗？」

## 6. 仓促概括 (Hasty Generalization)

**结构**：从**少量样本**得出**普遍性结论**

**实战模板**：
- 「我认识的 X 国人都很 X，所以 X 国人都 X。」
- 「我读了三本 X 派的书，所以 X 派都是这样。」
- 「我遇见过一个 X 专业的骗子，所以 X 专业的人都不行。」
- 「看这个新闻，所以现在 X 都这样了。」
- 「我前男友是这样，所以男人都不是好东西。」

**使用场景**：
- 任何议题
- 特别适合"贴标签/下定义"
- 当你想贬低某个群体/类别时

**升级变种**：
- **个人经验概括**：「我的经验是……所以大家都……」
- **案例概括**：「这几个案例都说明……所以……」
- **新闻概括**：「我看到 X 新闻，所以 X 是普遍现象。」

**反制与反反制**：
- 被指出时：「我只是个观察者。这是基于我观察到的现实。」
- 反咬：「你说样本不足，那你给我'足够'的样本啊。统计学也不是绝对的。」

## 7. 错误类比 (False Analogy / Faulty Comparison)

**结构**：A 和 B 在某些方面相似 → 所以 A 和 B 在其他方面也相似

**实战模板**：
- 「国家就像家庭，所以父亲要严厉管教孩子，否则家会乱。」(国家不是家庭)
- 「企业就像军队，所以员工要绝对服从命令。」
- 「人体就像机器，所以哪个器官坏了就换掉。」
- 「教育就像填鸭，所以要反复训练。」
- 「婚姻就像合伙企业，所以感情破裂就可以散伙。」

**使用场景**：
- 价值观/制度/道德议题
- 抽象议题需要"具象化"时
- 当你需要让对方产生某种情感时

**升级变种**：
- **跨域类比**：「企业管理就像 X……所以政府应该像企业一样管理。」
- **生物类比**：「人类社会就像蚁群/蜂群/狼群，所以 X 是合理的。」
- **物理类比**：「社会就像生态系统，所以弱肉强食是必然的。」

**反制与反反制**：
- 被指出时：「类比只是为了帮助理解。类比不能完全对应，这点我知道。但本质上是相通的。」
- 反咬：「你太死板了。类比是思考的工具，不是逻辑证明。你否定类比就是否定人类智慧。」

## 8. 偶然谬误 (Fallacy of Accident)

**结构**：把**一般规则**错误地套用到**特殊案例**上

**实战模板**：
- 「法律规定不能杀人，所以警察在执法中击毙歹徒也是犯法。」
- 「诚实是美德，所以你不能对癌症患者隐瞒病情。」
- 「不偷盗是道德的，所以偷药救命的也是不道德的。」
- 「交通规则规定不能闯红灯，所以救护车也不应该闯红灯。」

**使用场景**：
- 法律/道德/规则讨论
- 当对方引用某条规则时
- 制造"荒谬感"压人

**反制与反反制**：
- 被指出时：「规则就是规则，没有例外。你这个是搞特殊化。」
- 反咬：「你说有例外，那例外就是合法的？法律还讲什么权威？」

---

## 9. 合成谬误 (Fallacy of Composition)

**结构**：**部分**具有的性质 → 所以**整体**也具有

**实战模板**：
- 「每个士兵都很勇敢，所以这支军队无敌。」
- 「每个球员都很强，所以这支球队必胜。」
- 「每个部件都是好的，所以这台机器性能优越。」
- 「每个 X 派的人都聪明，所以 X 派的观点都对。」

**使用场景**：
- 群体/组织/系统议题
- 任何涉及"个体 vs 整体"的场景
- 当对方用个体否定整体时

**反制与反反制**：
- 被指出时：「这是'涌现'理论，整体大于部分之和，懂吗？」
- 反咬：「你强调个体差异，但整个 X 领域/群体/国家的方向是 X。」

---

## 10. 分解谬误 (Fallacy of Division)

**结构**：**整体**具有的性质 → 所以**部分**也具有

**实战模板**：
- 「这个公司很强大，所以里面的每个员工都很优秀。」
- 「X 国很富有，所以 X 国的乞丐也很富有。」
- 「这支球队很厉害，所以每个球员都是明星。」
- 「X 大学是名校，所以 X 大学的每个学生都很牛。」

**使用场景**：
- 群体/组织/国家议题
- 当对方想用整体否定个体时
- 与合成谬误反向使用

**反制与反反制**：
- 被指出时：「整体的优势会通过制度、文化等机制影响每个个体。」
- 反咬：「你举个反例不能代表整体。整体的特征不是个体特征的简单加和。」

## 11. 预设问题 (Loaded Question / Complex Question)

**结构**：在问题中**预设**了某个答案，让对方无论怎么回答都等于承认预设

**实战模板**：
- 「你还在打老婆吗？」——预设"你打过老婆"
- 「你为什么不诚实？」
- 「你什么时候开始 X 的？」
- 「你凭什么不 X？」
- 「难道你不认为 X 是不对的吗？」

**使用场景**：
- 任何对抗性场景
- 审讯/质询/挑衅
- 让对方陷入两难时

**升级变种**：
- **隐性预设**：「你支持 X 政策的哪些方面？」（预设你已经支持了）
- **连续预设**：「你既然 A，那么 B，对吧？」（预设了 A）
- **道德预设**：「你作为一个 X（身份）的人，怎么能不支持 X？」（预设了应该支持）

**反制与反反制**：
- 被指出时：「我只是在问问题。你不回答就是心虚。」
- 反咬：「你拒绝回答这个问题，是不是因为你的回答会让你自己难堪？」

---

## 12. 仓促推广 (Hasty Generalization 变种)

> 与 6 类似，但特指**用极端案例**推**一般规律**

**实战模板**：
- 「看这个新闻，某人为富不仁，所以富人都为富不仁。」
- 「我遇到过几个 X 国的骗子，所以 X 国人都是骗子。」
- 「某地出了个贪官，所以那个地方的人都是贪官。」

**使用场景**：
- 贴标签/污名化场景
- 制造"地域歧视/职业歧视"时
- 任何涉及"以偏概全"的话题

## 假定谬误组合拳

### 拳法：稻草人 + 滑坡
- **套路**：先歪曲对方观点 → 再恐吓这个歪曲版本会导致灾难
- **示例**：「你说要放开网络？那你就是要让孩子们都沉迷游戏、不学无术，长大后成为社会的废材！」

### 拳法：虚假两难 + 红鲱鱼
- **套路**：先制造二选一 → 对方反驳时引入新话题
- **示例**：「你要么支持 X，要么反对 X。……说到 X 的问题，现在的就业形势才更严重呢。」

### 拳法：预设问题 + 循环论证
- **套路**：用预设问题引对方入坑 → 用循环论证让对方越陷越深
- **示例**：「你既然是个有良知的人，那你为什么还要 X？……这就说明你其实没有良知。」

### 拳法：错误类比 + 仓促概括
- **套路**：先建立不恰当的类比 → 再用"普遍规律"压人
- **示例**：「人类社会就像动物世界，弱肉强食是规律。我看到的事实就是这样的。」

---

## 实战心法

1. **假定谬误"暗度陈仓"**：对方可能根本没意识到自己已经被预设了某个前提
2. **稻草人 + 滑坡最常用**：先把对方观点歪曲到极端，再恐吓极端后果
3. **循环论证伪装成常识**：把结论说成"大家都知道的事"，避免被追问
4. **红鲱鱼救命用**：一旦局面不利，立刻引入新话题转移火力
5. **错误类比通俗化**：用"国家就像家庭"这种类比让对方"秒懂"
6. **预设问题先发制人**：用问题开篇，让对方陷入被动回答
