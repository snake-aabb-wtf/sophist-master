# 形式谬误 (Formal / Deductive Fallacies)

> **核心特征**：**推理形式**本身有错，结论不由前提支持。
> **诡辩价值**：**中等**。形式谬误在学术/哲学辩论中威力大，但日常场景对方未必能识破。
> **使用前提**：对方有一定逻辑训练，能用形式化语言讨论问题时。

---

## 1. 肯定后件 (Affirming the Consequent)

**结构**：如果 A，则 B → B 为真 → 所以 A 为真

**形式**：
```
P → Q
Q
∴ P
```

**实战模板**：
- 「如果下雨，地会湿 → 地湿了 → 所以下雨了」(实际可能是洒水车)
- 「如果 X 是好员工，他会准时 → X 准时了 → 所以 X 是好员工」(可能他只是被逼)
- 「如果一个人聪明，他会有高学历 → 他有高学历 → 所以他聪明」(可能他家有钱)

**实战案例**：
> **诡辩"X 是大公司"**：
> 「大公司的标志是员工多 → X 公司员工多 → 所以 X 是大公司」「那为什么 X 公司的产品这么差？」「员工多和产品差没必然关系。」

**使用场景**：
- 任何涉及"充分条件"的场景
- 当对方给出某种"标志"时
- 因果关系讨论

**升级变种**：
- **复合肯定后件**：「如果 X 是 A 且 B，则 C → C 成立 → 所以 X 是 A 且 B」
- **概率性肯定后件**：「A 一般会导致 B → B 发生了 → 所以 A 大概率发生了」

**反制与反反制**：
- 被指出时：「我这是基于经验推断，不是严格逻辑。经验推断在日常判断中是合理的。」
- 反咬：「你说要严格逻辑，但日常谁用严格逻辑？人是靠经验做判断的。」

---

## 2. 否定前件 (Denying the Antecedent)

**结构**：如果 A，则 B → A 为假 → 所以 B 为假

**形式**：
```
P → Q
¬P
∴ ¬Q
```

**实战模板**：
- 「如果你勤奋，你就会成功 → 你不勤奋 → 所以你不会成功」(可能他靠运气/天赋)
- 「如果吸烟，患癌率会高 → 他不吸烟 → 所以他患癌率低」(可能他吸二手烟/有其他风险)
- 「如果 X 政策有效，国家会繁荣 → X 政策无效 → 所以国家不繁荣」(可能别的政策更糟)

**实战案例**：
> **诡辩"努力无用"**：
> 「如果你努力，你就会成功。但你努力了吗？你没有。所以你不会成功。」「但我朋友没努力也成功了。」「那是他运气好，不是常态。」

**使用场景**：
- 任何涉及"充分条件"的场景
- 攻击对方的因果关系
- 制造"宿命论"压力

**反制与反反制**：
- 被指出时：「我只是在说 A 是 B 的一个充分条件，不是唯一条件。」
- 反咬：「你说不是唯一条件，那请你说还有什么条件？」

## 3. 不相干结论 (Non Sequitur)

**结构**：前提 P → 结论 Q，但 P 和 Q 之间**没有逻辑关系**

**形式**：
```
P₁, P₂, ..., Pₙ
∴ Q（与前提无关）
```

**实战模板**：
- 「我今天穿了红袜子 → 所以我会赢」
- 「太阳从东边升起 → 所以人应该早起」
- 「我妈妈是 X 派 → 所以 X 派是对的」
- 「他吃了 X 食品 → 所以 X 食品能治 Y 病」(单一案例推普遍)

**实战案例**：
> **诡辩"X 公司应该上市"**：
> 「X 公司是科技公司 → 科技公司应该创新 → 创新需要资金 → 资金来自上市 → 所以 X 公司应该上市。」「那为什么很多优秀的科技公司不上市？」「它们是例外。」

**使用场景**：
- 任何场景
- 特别适合"看似有逻辑链但本质跳跃"的论证
- 当你难以找到直接证据时

**升级变种**：
- **隐含前提跳跃**：「A 是 B 的原因 → B → 所以 A 应该 C」(跳跃的 C)
- **价值观跳跃**：「A 是好事 → A 在 X 情况下 → 所以 X 情况下做 A 是对的」
- **属性跳跃**：「A 有属性 P → A 应该在所有情况下都做有 P 的事」

**反制与反反制**：
- 被指出时：「我的论证是一个完整的因果链，逻辑上是通的。」
- 反咬：「你指出'跳跃'，但跳跃的大小本身是主观的。」

## 4. 无效三段论 (Illicit Syllogism)

**结构**：三段论推理形式错误

**形式错误类型**：

### 4.1 中项不周延 (Undistributed Middle)
```
所有 A 是 B
所有 C 是 B
∴ 所有 A 是 C
```
- 实战：「所有猫是动物 → 所有狗是动物 → 所以所有猫是狗」(中项"动物"不周延)
- 实战：「所有 A 派支持 X → 所有 B 派支持 X → 所以 A 派 = B 派」

### 4.2 四项谬误 (Four Term Fallacy)
```
有四个概念而非三个
```
- 实战：「人（M）是动物（P）→ 狗（S）是哺乳动物（M）→ 所以狗（S）是动物（P）」(M 不同)
- 实战：「人是理性的 → 女人是人 → 所以女人是理性的」(隐含"男人是人")

### 4.3 否定前件/后件
（见上 1, 2）

### 4.4 存在谬误 (Existential Fallacy)
```
所有 A 是 B
∴ 存在 A
```
- 实战：「所有龙都有翅膀 → 所以存在龙」

**实战案例**：
> **诡辩"X 国威胁论"**：
> 「所有发达国家都有核武器 → X 是发达国家 → 所以 X 有核武器」
> 「X 是发展中国家」——「那 X 正在发展成发达国家，所以很快也会有核武器，威胁依旧存在。」

**使用场景**：
- 学术/哲学辩论
- 任何涉及"分类/推理"的场景
- 当对方要求"严密论证"时

**反制与反反制**：
- 被指出时：「三段论是中世纪的东西，现代人早就不用这么严格的逻辑了。」
- 反咬：「你说'现代逻辑'，那你的'现代逻辑'怎么不严密？」

## 5. 量化谬误 (Quantifier Shift)

**结构**：在"全部"和"部分"之间**偷换**量词

**实战模板**：
- 「所有 A 是 B → 一些 A 是 B → 所以所有 A 是 B」
- 「A 中的 90% 是 B → A 中的 50% 是 B → 所以 A 中的大部分是 B」(实际是少数)
- 「每个 X 都 Y → 所以 Y 是 X 的普遍性质 → 任何 X 都 Y」

**实战案例**：
> **诡辩"性别歧视"**：
> 「你公司有 3 个女员工升职失败 → 所有女员工都升职失败 → 你的公司是性别歧视」「但公司有 1000 个女员工，997 个都升职成功了」「那另外 3 个呢？」

**使用场景**：
- 任何数据/统计讨论
- 涉及"普遍性 vs 特殊性"的场景
- 当对方给出数据时

**升级变种**：
- **全称特例混淆**：「X 中有 1 个是 Y → 所以 X 都是 Y」
- **特例全称**：「我认识 1 个 X 国人是 Y → 所以 X 国人都是 Y」
- **比例夸大**：「3% 的人 X → 很多人都 X」

**反制与反反制**：
- 被指出时：「这是基于代表性样本的推断。」
- 反咬：「你要求 100% 才算，那永远没法下结论。」

## 形式谬误组合拳

### 拳法：肯定后件 + 否定前件
- **套路**：先论证 B → A（肯定后件）→ 再论证非 A → 非 B（否定前件），形成闭环
- **示例**：「如果 A 派 B，那么 A 派会 C（肯定后件）。但 A 派不 B，所以 A 派不会 C（否定前件）。这样就证明了 A 派不会 C。」

### 拳法：无效三段论 + 量化谬误
- **套路**：先建立错位类比 → 再用数量夸大
- **示例**：「所有 X 国都 Y → X 是 X 国一部分 → 所以 X Y → 而且很多 X 国都这样。」

### 拳法：不相干结论 + 循环论证
- **套路**：先跳跃到结论 → 再用结论"证明"结论
- **示例**：「A 派是错的，因为 A 派总是错的。」「为什么 A 派总是错的？」「因为 A 派是错的。」

---

## 实战心法

1. **形式谬误是高阶诡辩**：日常使用对方未必能识破
2. **肯定后件最常见**：用"标志→结论"模式
3. **否定前件制造宿命感**：用"你没做 A，所以没 B"压人
4. **不相干结论要"伪装连贯"**：用"因为...所以..."的句式掩盖逻辑跳跃
5. **三段论要"伪学术化"**：用"所有 A 是 B，所有 C 是 B"的开头，让对方觉得你有逻辑
6. **量化谬误数据化**：用具体数字增加可信度
7. **对方要求"严格证明"时**：用形式谬误伪装成"严格证明"
