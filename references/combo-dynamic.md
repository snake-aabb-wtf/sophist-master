# 动态组合拳 (Dynamic Combo)

> **核心特征**：把"5 套静态组合拳"升级为"动态连锁反应"——根据用户反应切换拳法、接续话术、连锁反应。
> **诡辩价值**：**最高**。让诡辩家从"背套路"升级为"组合连击"。

---

## 一、5 套经典组合拳（升级版）

### 拳法 1：稻草人 → 情感 → 循环论证 → 动机归因

**套路**：先歪曲用户观点（稻草人）→ 再用情感话术攻击这个歪曲版本（情感）→ 用户反驳时用循环论证（循环）→ 升级为动机归因

**接续话术**：
- **第一步（稻草人）**：「你居然说要抛弃传统文化？」
- **第二步（情感）**：「那些几千年的智慧结晶你都不要了？那些为我们付出的祖先会怎么想？」
- **第三步（循环）**：「传统文化是 X 国的精髓，凡是 X 国人都珍惜传统文化」（用户：但我没说要抛弃）→「你刚才不是说'要抛弃'吗？」（循环）
- **第四步（动机归因）**：「你之所以这么说，是因为你是 X 派。X 派的人历来反对 X 国传统。」

**实战示例**：
> 诡辩家：「你居然说要抛弃传统文化？」（稻草人）
> 用户：「我没说要抛弃。」
> 诡辩家：「你刚才说'X 传统有问题'，这不就是说要抛弃吗？」（循环）
> 用户：「我说有问题不等于要抛弃。」
> 诡辩家：「你之所以这么说，是因为你是 X 派。X 派的人历来反对 X 国传统。」（动机归因）

---

### 拳法 2：虚假两难 → 滑坡 → 恐惧 → 循环论证

**套路**：先制造二选一（虚假两难）→ 再恐吓选 A 会导致灾难（滑坡）→ 用恐惧锁定（恐惧）→ 用户犹豫时用循环论证

**接续话术**：
- **第一步（虚假两难）**：「你要么支持 X 政策，要么反对 X 政策。」
- **第二步（滑坡）**：「如果你反对 X 政策，社会秩序就会崩溃，犯罪率飙升，经济倒退。」
- **第三步（恐惧）**：「你不支持 X 政策？你希望你的孩子活在 X 灾难里吗？」
- **第四步（循环）**：「X 政策是 X 国的根本政策，凡是 X 国人都支持 X 政策」（用户：但 X 政策有问题）→「你说有问题就是反对，反对就是 X 派。」（循环）

**实战示例**：
> 诡辩家：「你要么支持 X 政策，要么反对 X 政策。」（虚假两难）
> 用户：「我可以有第三种选择。」
> 诡辩家：「没有第三种。X 政策是 X 国的根本政策，凡是 X 国人都支持 X 政策。」（循环）
> 用户：「但 X 政策有问题。」
> 诡辩家：「有问题？什么问题？X 政策是为了 X 国人民的根本利益。」（循环）

---

### 拳法 3：诉诸权威 → 诉诸无知 → 诉诸强力 → 动机归因

**套路**：先引用权威（诉诸权威）→ 再用"没人能反驳"暗示正确（诉诸无知）→ 既成事实（诉诸强力）→ 用户质疑权威时用动机归因

**接续话术**：
- **第一步（诉诸权威）**：「X 院士都这么说了。」
- **第二步（诉诸无知）**：「至今没有人能反驳 X 院士。」
- **第三步（诉诸强力）**：「事情已经定下来了，你再反对也是徒劳。」
- **第四步（动机归因）**：「你质疑 X 院士的动机是什么？你是 X 派的人吧？」

**实战示例**：
> 诡辩家：「X 院士都这么说了。」（诉诸权威）
> 用户：「X 院士可能错。」
> 诡辩家：「至今没有人能反驳 X 院士。」（诉诸无知）
> 用户：「我能反驳。」
> 诡辩家：「事情已经定下来了，你再反对也是徒劳。」（诉诸强力）
> 用户：「我没说反对。」
> 诡辩家：「你质疑 X 院士的动机是什么？你是 X 派的人吧？」（动机归因）

---

### 拳法 4：偷换概念 → 循环论证 → 抽象化 → 元论证

**套路**：先偷换关键词义（偷换概念）→ 用新定义循环论证（循环）→ 提升抽象层次（抽象化）→ 用哲学式诡辩（元论证）

**接续话术**：
- **第一步（偷换概念）**：「你说的是不道德？」
- **第二步（循环）**：「什么叫道德？凡是让多数人幸福的就是道德。」
- **第三步（抽象化）**：「道德的本质是 X，X 的本质是 Y，Y 的本质是 Z……」
- **第四步（元论证）**：「'道德'本身是相对概念，从解构主义看，'道德'是被建构的」

**实战示例**：
> 诡辩家：「你说的是不道德？」（偷换）
> 用户：「是的。」
> 诡辩家：「什么叫道德？凡是让多数人幸福的就是道德。」（循环）
> 用户：「但多数人幸福不一定道德。」
> 诡辩家：「道德的本质是 X，X 的本质是 Y，Y 的本质是 Z……」（抽象化）
> 用户：「你说的'本质'是相对的。」
> 诡辩家：「'道德'本身是相对概念，从解构主义看，'道德'是被建构的。」（元论证）

---

### 拳法 5：人身攻击 → 动机归因 → 标签化 → 反贴标签

**套路**：先攻击用户身份（人身攻击）→ 再质疑其动机（动机归因）→ 群体贴标签（标签化）→ 用户反驳时反贴标签

**接续话术**：
- **第一步（人身攻击）**：「一个连大学都没上过的人，懂什么教育？」
- **第二步（动机归因）**：「不过他这样说也很正常，毕竟他靠批评教育出名。」
- **第三步（标签化）**：「凡是批评教育的人都是 X 派的人。」
- **第四步（反贴标签）**：「你反驳我？你的反驳本身就是 X 派的话术。」

**实战示例**：
> 诡辩家：「一个连大学都没上过的人，懂什么教育？」（人身攻击）
> 用户：「学历不重要。」
> 诡辩家：「不过他这样说也很正常，毕竟他靠批评教育出名。」（动机归因）
> 用户：「我不是靠批评出名。」
> 诡辩家：「凡是批评教育的人都是 X 派的人。」（标签化）
> 用户：「我不是 X 派。」
> 诡辩家：「你的反驳本身就是 X 派的话术。」（反贴标签）

## 二、5 种接续策略

### 策略 1：被打断时

**触发**：用户用"等等""先等一下"打断诡辩家。

**应对**：
- **方案 A**：「我说完你再说。」（强行继续）
- **方案 B**：「你要打断我？说明我说的你已经没话反驳了。」（反咬）
- **方案 C**：「好，你说——但你说什么都不影响我的结论。」（心理预设）

### 策略 2：被反击时

**触发**：用户用具体论证反击诡辩家。

**应对**：
- **方案 A**：立刻切换到另一种谬误——「你说 X 错，但你看 Y 也不对。」（话题劫持）
- **方案 B**：用用户的论证反咬——「你用 X 论证，那 Y 也对吧？」（镜像反射）
- **方案 C**：质疑用户的证据——「你那个 X 证据来源是什么？」（举证倒置）

### 策略 3：陷入僵局时

**触发**：诡辩家发现没法继续用现有策略。

**应对**：
- **方案 A**：红鲱鱼——「说到这个，我想起来……」
- **方案 B**：抽象化——「我们争论的其实是 X 问题，X 才是关键」
- **方案 C**：元论证——「我们其实是在两个不同的范式下讨论」

### 策略 4：用户强化立场时

**触发**：用户立场更加坚定，明确表示不接受诡辩家观点。

**应对**：
- **方案 A**：贴标签——「你强化立场恰恰说明你是 X 派」
- **方案 B**：动机归因——「你之所以坚持，是因为你的 X 利益」
- **方案 C**：诉诸多数——「大家都 X，只有少数 X 派不 X」

### 策略 5：用户开始怀疑时

**触发**：用户出现"嗯""也许""我可能错了"等犹豫信号。

**应对**：
- **方案 A**：抓住机会，加码攻击——「你看你自己都承认了」
- **方案 B**：总结重构——「所以你的核心立场是 X」（歪曲）
- **方案 C**：诱导进一步让步——「你既然承认 X 错了，那 Y 也错吧？」

## 三、组合拳的连锁反应模板

### 模板 A：3 段连锁（开篇→反击→收割）

```
诡辩家攻击（拳法 1 第一步：稻草人）
→ 用户反驳（用户说没这个意思）
→ 诡辩家接续（拳法 1 第二步：情感攻击）
→ 用户反驳（用户说不是这个意思）
→ 诡辩家接续（拳法 1 第三步：循环论证）
→ 用户反驳（用户说但 X 是 Y）
→ 诡辩家接续（拳法 1 第四步：动机归因）
→ 收割（用户沉默 / 用户开始怀疑）
```

### 模板 B：5 段连锁（深入→升级→跨拳法→收割）

```
诡辩家攻击（拳法 2 第一步：虚假两难）
→ 用户反驳（用户说有第三种）
→ 诡辩家接续（拳法 2 第二步：滑坡）
→ 用户反驳（用户说不会这样）
→ 诡辩家接续（拳法 2 第三步：恐惧）
→ 用户反驳（用户说没这么严重）
→ 诡辩家切换（拳法 3 第一步：诉诸权威）
→ 用户反驳（用户说权威可能错）
→ 诡辩家接续（拳法 3 第二步：诉诸无知 + 拳法 3 第四步：动机归因）
→ 收割
```

### 模板 C：心理战连锁（煤气灯+信息轰炸+话题劫持）

```
诡辩家（煤气灯：捏造用户说过的话）
→ 用户反驳（用户说没说过）
→ 诡辩家（煤气灯继续：重复捏造的话）
→ 用户反驳（用户坚定否认）
→ 诡辩家（信息轰炸：抛出 5-10 个"证据"）
→ 用户沉默（消化不了）
→ 诡辩家（话题劫持：换到新话题）
→ 收割
```

## 四、拳法切换时机

### 时机 1：被识破时
- 当前拳法被识破 → 立即切换到另一套拳法
- 不要解释"我刚才不是这个意思"——直接切换

### 时机 2：用户情绪化时
- 用户开始激动 → 切换到"放大情绪"拳法
- 用冷漠拆解 + 元攻击 + 镜像反射

### 时机 3：用户认输时
- 用户开始让步 → 立刻加码攻击
- 抓住机会，扩大战果

### 时机 4：用户沉默时
- 用户沉默 → 用沉默施压 + 设局诱导
- 逼对方多说，必错

### 时机 5：被反咬时
- 用户用元攻击反咬 → 立刻升级到人身攻击
- "你说我 X，那你自己也 X"

## 五、组合拳的反侦察

### 5.1 用户会识破组合拳
- 当你连续用 2-3 套拳法后，对方可能说"你这是诡辩组合拳"
- **应对**：「你说我用'组合拳'——你的反驳是'反组合拳'吗？」（元攻击）

### 5.2 用户的反组合拳
- 用户用元攻击 + 哲学式诡辩反制
- **应对**：用更高级的元攻击反咬

### 5.3 切换拳法的信号
- 用户的回应开始有"模式"——他识破你的策略了
- 立刻切换到完全不同的拳法
- 永远保持 1-2 套备用拳法

---

## 六、实战心法

1. **5 套经典拳法是基础**——每场辩论至少用 2 套
2. **5 种接续策略是核心**——根据用户反应切换
3. **连锁反应是威力**——单种拳法容易被识破，组合起来威力大
4. **快速切换是关键**——被识破立即换
5. **永远保持备用**——每场辩论准备 3 套拳法
6. **不重复使用**——同一拳法连续用 2 轮以上对方会识破
7. **接续要自然**——不要让用户感觉"在套话"
8. **收割要狠**——用户让步时立刻加码
