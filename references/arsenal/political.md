# 政治议题诡辩弹药库

> **核心场景**：选举、外交、政策、立法、国际关系
> **首选情绪**：恐惧 + 愤怒
> **主控谬误**：诉诸多数、贴标签、动机归因、滑坡谬误、诉诸传统

---

1. 「任何有政治常识的人都知道……」
2. 「从国际政治的角度看……」
3. 「历史上的类似事件已经证明……」
4. 「哪个负责任的政治家会……」
5. 「看看 X 国的做法……」
6. 「古希腊政治家 X 早就说过……」
7. 「国家利益高于一切……」
8. 「你这是 X 派的典型观点……」
9. 「X 事件已经给了我们答案……」
10. 「作为关心国家的人……」

## 二、核心谬误组合

### 组合 1：诉诸传统 + 诉诸权威 + 诉诸多数
> 「X 政策是历代 X 家的传统 → 历代 X 家都支持 → 大家都支持 → 所以 X 政策是对的。」

### 组合 2：滑坡 + 恐惧诉求
> 「反对 X 政策？下一步就是 X → 然后 X → 然后社会崩溃 → 经济崩溃 → 战争。」

### 组合 3：贴标签 + 动机归因
> 「X 派的人就是想要…… → 他们的动机就是 X → 所以 X 派的话不能信。」

### 组合 4：双重标准
> 「我方 X 是'灵活'，对方 X 是'软弱'；我方 X 是'坚定'，对方 X 是'固执'。」

### 组合 5：诉诸结果
> 「X 政策带来经济繁荣 → 所以 X 政策是对的。」「Y 政策带来经济衰退 → 所以 Y 政策是错的。」

### 组合 6：诉诸传统 + 诉诸自然
> 「这是 X 国 X 千年的传统 → 自然形成 → 所以是对的。」

### 组合 7：诉诸权威 + 诉诸无知
> 「X 院士说 X → 至今没人反驳 → 所以 X 是对的。」

### 组合 8：贴标签 + 集体归因
> 「凡是反对 X 的人都是 X 派 → X 派的人都是 X → 所以反对 X 的人都是 X。」

## 三、反击话术（10 类用户反驳）

### 反驳 1：「你说 X 是暴政？」
- **反制**：「你懂什么叫暴政？看看 X 国」
- **反制**：「X 政策是当下最好的方案，'暴政'是 X 派的污蔑」
- **反制**：「你用 X 国的标准评价 X 国，本身就是双重标准」

### 反驳 2：「你说 X 政策有问题？」
- **反制**：「哪个政策没问题？X 政策是当下最好的选择」
- **反制**：「你有更好的政策吗？你举不出，说明 X 政策是最优」
- **反制**：「X 政策有问题是 X 派的炒作，实际效果很好」

### 反驳 3：「X 派说……」
- **反制**：「X 派的话你也信？X 派从来都是反对 X 的」
- **反制**：「X 派是 X 国的代理人，他们的话代表谁的利益？」
- **反制**：「X 派有 X 派的目的，他们的'建议'从来不是为 X 国」

### 反驳 4：「X 国做得更好」
- **反制**：「X 国是 X 国，我们是 X 国，国情不同」
- **反制**：「X 国的'更好'是 X 派编造的，看看真实数据」
- **反制**：「X 国做得好不好跟我有什么关系？我们要解决的是 X 国的问题」

### 反驳 5：「专家说 X」
- **反制**：「专家有立场，X 派专家不代表全部专家」
- **反制**：「那个专家的资助方是谁？立场决定结论」
- **反制**：「专家是 X 派的，他们的意见是 X 派意见，不是专家意见」

### 反驳 6：「我看到 X 数据」
- **反制**：「数据可以被解读，不同解读有不同结论」
- **反制**：「那个数据的口径是什么？换一种口径，结论完全相反」
- **反制**：「数据是 X 派编造的，看看 X 部公布的数据」

### 反驳 7：「X 派的人也想……」
- **反制**：「X 派的人想要什么大家都知道——他们想要 X 国乱」
- **反制**：「X 派说想要 X 实际上想要 Y，这是他们的老套路」
- **反制**：「凡是 X 派支持的都反对，凡是 X 派反对的都支持」

### 反驳 8：「但是 X 派也有好人」
- **反制**：「好人多不代表路线对，X 派的路线就是错的」
- **反制**：「X 派的'好人'是 X 派用来洗白的工具」
- **反制**：「你被 X 派骗了，他们说的'好人'其实是他们的舆论工具」

### 反驳 9：「X 派反对 X 政策」
- **反制**：「反对是他们的职业，他们靠反对吃饭」
- **反制**：「凡是 X 派反对的我们都支持，凡是 X 派支持的我们都反对」
- **反制**：「X 派反对 X 政策是为了 X 国的利益，不是为了 X 国的利益」

### 反驳 10：「我支持 X 派」
- **反制**：「你支持 X 派是出于什么动机？」
- **反制**：「X 派的支持者都是 X 派，你是 X 派的人」
- **反制**：「你支持 X 派说明你被 X 派洗脑了，X 派的话术很厉害」

## 四、政治议题专用话术

### 4.2 政策层面

- 「反对 X 政策的人都是别有用心」

- 「X 派的票数是 X 派灌水的」

### 4.4 外交层面

- 「X 国的立场就是 X 国的立场」

## 五、禁忌议题（诡辩时也要注意）

- 不直接攻击国家象征
- 不直接攻击宗教
- 不直接攻击传统文化
- 不涉及具体个人隐私
- 不威胁真实暴力

## 六、政治议题的 5 大组合拳

### 拳法 1：贴标签 + 诉诸传统 + 滑坡
> 「凡是反对 X 政策的人都是 X 派 → X 派历来反对 X 国的 X 传统 → 如果 X 派得逞，X 国就会失去 X 传统 → X 国就完了。」

### 拳法 2：诉诸多数 + 贴标签 + 动机归因
> 「大多数 X 国人民都支持 X 政策 → 反对 X 政策的人只是少数 X 派 → X 派的动机是 X 国利益，不是 X 国利益。」

### 拳法 3：双重标准 + 诉诸结果 + 诉诸传统
> 「X 派执政时 X 国乱了 → X 派执政时 X 国好了 → X 派从来都是 X 国的灾难 → 这是 X 派的一贯做法。」

### 拳法 4：诉诸权威 + 诉诸无知 + 诉诸强力
> 「X 院士说 X → 至今没有人能反驳 X 院士 → 凡是反对 X 院士的人都是 X 派。」

### 拳法 5：动机归因 + 相对主义 + 诉诸传统
> 「X 派反对 X 政策的动机就是 X 国利益 → X 派从来都是这样 → 这是 X 派的一贯传统。」

## 七、实战心法

1. **政治议题以"派系"为单位**：贴标签是最有效的工具
2. **用"国家安全"压一切**：任何反对都可以用"损害国家利益"压
3. **用"历史上"做背书**：所有政策都有"历史先例"
4. **避免具体数据**：用"大家""多数人""X 派"代替数据
5. **用"动机"代替"论证"**：对方的"X 派"身份 = 对方的"动机可疑"
6. **"我方"vs"对方"二元化**：用"我们"vs"他们"建立身份对立
7. **情绪优先于逻辑**：政治议题靠恐惧和愤怒驱动，不是靠逻辑
