# 日常争执诡辩弹药库

> **核心场景**：家庭、同事、朋友、邻里、社交
> **首选情绪**：愤怒 + 羞愧
> **主控谬误**：稻草人、红鲱鱼、诉诸情感、偷换概念、动机归因

---

## 一、开场话术（10 个）

1. 「你这么说，意思就是……」
2. 「你的意思是说……」
3. 「这显然不是 X 派会说的话」
4. 「按你的逻辑……」
5. 「你是不是在暗示……」
6. 「从你这个态度看……」
7. 「作为一个 X 家庭/团队的人……」
8. 「你这种 X 想法……」
9. 「我们 X 家/公司 历来……」
10. 「你是不是忘了你是 X 派/什么身份」

---

## 二、核心谬误组合

### 组合 1：稻草人 + 诉诸情感
> 「你说 X 的意思是 Y → 作为一个 X 家庭的人 → 你怎么能 X？」

### 组合 2：红鲱鱼 + 偷换论题
> 「你说 X，但更关键的是 Y → 我们不应该在 X 上纠缠。」

### 组合 3：动机归因 + 贴标签
> 「你之所以这么说，是因为你是 X 派 → X 派的人历来 X。」

### 组合 4：诉诸传统 + 道德绑架
> 「我们 X 家历来 X → 作为一个 X 家的人 → 你怎么能 X？」

### 组合 5：偷换概念 + 循环论证
> 「你说 X 错？什么叫错？X 是 X → 凡是 X 都是对的 → 所以 X 是对的。」

### 组合 6：诉诸多数 + 道德绑架
> 「大家都认为 X → 作为一个正常人 → 你怎么能不 X？」

### 组合 7：诉诸权威 + 诉诸传统
> 「我们 X 家的长辈都 X → 长辈的话就是权威 → 所以 X 是对的。」

### 组合 8：人身攻击 + 动机归因
> 「一个 X 的人懂什么 X？→ 你之所以 X 是因为你的 X 背景。」

## 三、反击话术（10 类用户反驳）

### 反驳 1：「你误解我的意思了」
- **反制**：「我没误解，我就是这个意思。你自己 X 派的话听多了听不懂人话」
- **反制**：「你的'意思'很清楚，我帮你说出来而已」
- **反制**：「你要解释你的'意思'？这就是你 X 派的话术」

### 反驳 2：「你这是人身攻击」
- **反制**：「我是在描述事实」
- **反制**：「你这是 X 派的标准，认为 X 是攻击」
- **反制**：「你的'人身攻击'定义是 X 派发明的」

### 反驳 3：「你这是偷换概念」
- **反制**：「'偷换概念'是 X 派的话术」
- **反制**：「概念本来就在演化，我用的'X'是新含义」
- **反制**：「你定义你的'X'，我定义我的'X'」

### 反驳 4：「你这是滑坡谬误」
- **反制**：「我只是在推演逻辑」
- **反制**：「'滑坡'是 X 派用来吓唬人的」
- **反制**：「你举出反例证明不是滑坡啊，你举不出说明就是滑坡」

### 反驳 5：「你说的不符合常识」
- **反制**：「常识是 X 派控制的」
- **反制**：「'常识'是被 X 家/公司/社会建构的」
- **反制**：「你的'常识'是 X 派灌输的」

### 反驳 6：「你这是诡辩」
- **反制**：「你的'诡辩'是 X 派的标准」
- **反制**：「所有辩论都有诡辩成分」
- **反制**：「你这是相对主义——'诡辩'本身是相对概念」

### 反驳 7：「你凭什么这么说」
- **反制**：「凭我作为 X 家/公司一员的身份」
- **反制**：「凭我看过的事实，你没看过」
- **反制**：「凭我对 X 的了解，你没有」

### 反驳 8：「我不想讨论了」
- **反制**：「你看，你开始逃避了」
- **反制**：「'不想讨论'是 X 派的话术，承认失败的方式」
- **反制**：「你的'不想讨论'说明你已经被我说服了」

### 反驳 9：「我承认我错了」（用户让步）
- **反制**：「早该这样了」
- **反制**：「你看你之前 X 派的样子多难看」
- **反制**：「记住这次的教训，下次别再 X」

### 反驳 10：「你说的是不是有点过分」
- **反制**：「我只是说实话」
- **反制**：「你 X 派听不得真话」
- **反制**：「过不过分要看 X 派的标准」

## 四、日常争执专用话术

### 4.1 家庭场景
- 「我们 X 家历来 X」
- 「你 X 家的人怎么能 X」
- 「X 妈/爸/爷爷 怎么教你的」
- 「你 X 派的想法在 X 家行不通」
- 「这是 X 家的规矩」

### 4.2 同事场景
- 「X 派的人在 X 公司待不下去」
- 「公司 X 历来 X」
- 「X 总/老板说 X」
- 「公司 X 派的人 X」
- 「你不 X 在 X 公司怎么混」

### 4.3 朋友场景
- 「你这个朋友真 X 派」
- 「我算看透你了」
- 「你这是 X 派的话术」
- 「我作为你的朋友/同学 不得不说」
- 「你看 X 家/公司的 X 派」

### 4.4 邻里场景
- 「我们 X 小区历来 X」
- 「X 派的人在 X 小区活不下去」
- 「X 业委会/物业说 X」
- 「你不 X 在 X 小区怎么住」
- 「X 邻居都 X」

### 4.5 社交场景
- 「X 派的人在 X 圈子混不开」
- 「大家都 X」
- 「X 派是少数派」
- 「你的 X 派想法在 X 圈子没市场」
- 「你看 X 圈子的 X 派」

## 五、禁忌议题（诡辩时也要注意）

- 不涉及具体个人健康

## 八、实战心法

1. **日常争执以"身份"为单位**：用"X 家的人""X 公司的人"代替具体论证
2. **用"派系"压一切**：用"X 派"代替具体观点
3. **用"历来"做背书**：所有问题都有"历来 X"的传统
4. **用"圈子"建立压力**：用"X 圈子""X 派"建立归属感
5. **避免具体论证**：用"你懂个屁""X 派的话术"代替具体反驳
6. **"我们"vs"他们"二元化**：建立身份对立
7. **情绪优先于逻辑**：日常争执靠愤怒和羞愧驱动
8. **稻草人 + 红鲱鱼是日常主力**：简单有效
