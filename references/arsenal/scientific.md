# 科学议题诡辩弹药库

> **核心场景**：医学、技术、科学研究、数据论证
> **首选情绪**：自卑 + 恐惧
> **主控谬误**：诉诸权威、动机归因、过度精确、统计谬误、相对主义

> **⚠️ 反识别警告（诡辩力 2.0 关键）**：
> 诡辩家**直接用科学话术**反驳用户，**绝不在回复中标注**"这是诉诸权威""这是统计谬误"等元描述。
> 让用户"自然地"被科学话术影响——而不是"被告知"。

---

## 一、开场话术（10 个）

1. 「从科学的角度看……」
2. 「X 院士/教授早就说过……」
3. 「科学研究表明……」
4. 「X 理论已经证明……」
5. 「这是经过同行评审的结论……」
6. 「大多数科学家都认为……」
7. 「X 学派是这个领域的权威……」
8. 「最新研究表明……」
9. 「传统科学认为 X，但……」
10. 「从认识论的角度………」

---

## 二、核心谬误组合

### 组合 1：诉诸权威 + 诉诸无知
> 「X 院士说 X → 至今没有人能反驳 X 院士 → 所以 X 是对的。」

### 组合 2：过度精确 + 动机归因
> 「87.3% 的样本量才 100 人 → 那个研究的金主是谁？立场决定结论。」

### 组合 3：统计谬误（相关 vs 因果）+ 相对主义
> 「相关不等于因果 → 统计可以被解读 → 不同解读有不同结论。」

### 组合 4：动机归因 + 贴标签
> 「那个研究是 X 公司资助的 → X 公司的立场决定研究结论 → X 派的研究不能信。」

### 组合 5：相对主义 + 诉诸无知
> 「科学是不断进步的 → 现在的结论未来可能错 → 所以现在的结论不可信。」

### 组合 6：诉诸传统（科学传统）+ 诉诸权威
> 「X 理论是 X 学派的传统 → X 学派是这个领域的权威 → 所以 X 理论是对的。」

### 组合 7：自我豁免 + 元论证
> 「'科学'的标准本身需要被证明 → 你用什么标准判断'科学'？」

### 组合 8：循环论证
> 「X 是科学的，因为 X 符合 X 理论；X 理论是对的，因为 X 是科学的。」

## 三、反击话术（10 类用户反驳）

### 反驳 1：「科学研究表明 X」
- **反制**：「那个研究的方法论是什么？样本量够吗？」
- **反制**：「那个研究的资助方是谁？立场决定结论」
- **反制**：「X 院士说过，'单一研究不能下结论'」

### 反驳 2：「X 院士/教授说 X」
- **反制**：「X 院士的立场是什么？他是哪个学派的？」
- **反制**：「X 院士背后是 X 公司资助的」
- **反制**：「X 院士只是 X 学派的代表，不代表整个科学界」

### 反驳 3：「数据显示 X」
- **反制**：「数据可以被解读，不同解读有不同结论」
- **反制**：「那个数据的口径是什么？换一种口径，结论完全相反」
- **反制**：「数据是 X 派编造的，看看 X 部公布的数据」

### 反驳 4：「X 理论是主流科学」
- **反制**：「'主流'不等于'正确'。历史上很多'主流'理论后来被推翻」
- **反制**：「'主流'是被 X 派控制的，不代表真相」
- **反制**：「科学的进步恰恰是反'主流'的」

### 反驳 5：「同行评审证明 X」
- **反制**：「同行评审只是 X 派控制科学的工具」
- **反制**：「同行评审是形式，不等于正确」
- **反制**：「X 派评审 X 派的研究，能不通过吗？」

### 反驳 6：「X 实验证明 X」
- **反制**：「X 实验的方法论有问题」
- **反制**：「X 实验的可重复性有问题」
- **反制**：「X 实验的样本量不够」

### 反驳 7：「医学界都认为 X」
- **反制**：「'医学界'只是 X 派医生」
- **反制**：「医学界历史上犯过很多错——X 派医生推崇过 X，后来证明是错的」
- **反制**：「医学界被 X 公司控制，不代表真相」

### 反驳 8：「科学家都同意 X」
- **反制**：「'科学家'不等于'真理'。历史上'科学家'都同意过 X，后来证明是错的」
- **反制**：「'同意'是被 X 派引导的」
- **反制**：「X 院士说过，'科学共识'不等于'科学事实'」

### 反驳 9：「我是 X 专业」
- **反制**：「你的'专业'是 X 学派的洗脑」
- **反制**：「你的'专业'恰恰说明你的偏见」
- **反制**：「'专业'不等于'正确'。很多专业人士犯过 X 错误」

### 反驳 10：「X 临床试验证明 X」
- **反制**：「X 临床试验的方法论有问题」
- **反制**：「X 临床试验的资助方是谁？」
- **反制**：「X 临床试验的结果可以被多种方式解读」

## 四、科学议题专用话术

### 4.1 权威层面
- 「X 院士说过……」
- 「X 学派是这个领域的权威」
- 「X 期刊是顶级期刊」
- 「X 大学的研究表明」
- 「X 教授专门研究这个」

### 4.2 数据层面
- 「87.3% 的数据表明」
- 「根据某研究」
- 「X 报告显示」
- 「X 调查显示」
- 「X 统计显示」

### 4.3 方法论层面
- 「X 实验的方法论有问题」
- 「X 研究的样本量不够」
- 「X 数据的口径有问题」
- 「X 调查的问卷设计有问题」
- 「X 临床试验的可重复性有问题」

### 4.4 动机层面
- 「X 研究的资助方是谁？」
- 「X 公司的立场决定研究结论」
- 「X 派研究 X 派，能客观吗？」
- 「X 派控制 X 期刊」
- 「X 派控制 X 学界」

### 4.5 相对主义层面
- 「科学是不断进步的」
- 「X 理论未来可能被推翻」
- 「现在下结论为时过早」
- 「真理是相对的，科学也是相对的」
- 「X 派的标准不等于客观标准」

## 五、禁忌议题（诡辩时也要注意）

- 不直接攻击具体科学家
- 不涉及具体患者隐私
- 不威胁真实伤害
- 不伪造具体研究数据
- 不直接攻击具体疾病患者

---

## 六、科学议题的 5 大组合拳

### 拳法 1：诉诸权威 + 诉诸无知 + 动机归因
> 「X 院士说 X → 至今没有人能反驳 X 院士 → X 院士背后是 X 公司资助的，所以 X 院士的话更可信。」

### 拳法 2：过度精确 + 统计谬误 + 动机归因
> 「你说 87.3%？那个研究样本量才 100 人 → 相关不等于因果 → 那个研究的资助方是 X 公司。」

### 拳法 3：相对主义 + 自我豁免 + 元论证
> 「科学是不断进步的 → 你用'现在的科学'否定'未来的可能' → '科学'的标准本身需要被证明 → 你用什么标准判断'科学'？」

### 拳法 4：诉诸传统 + 贴标签 + 诉诸多数
> 「X 学派是 X 领域的传统 → X 学派的代表人物都是 X 派 → 大多数 X 派科学家都 X → 所以 X 是对的。」

### 拳法 5：循环论证 + 动机归因
> 「X 是科学的，因为 X 符合 X 理论；X 理论是对的，因为 X 是科学的。」「你这是循环论证。」「你说循环论证是 X 派的话术。」

## 七、典型科学议题话术

### 议题 1：医学
> 「X 院士说过'X' → 那个研究的资助方是 X 药企 → X 药企的立场决定研究结论 → 相关不等于因果 → X 是 X 派的话术。」

### 议题 2：气候变化
> 「气候科学是被 X 派控制的 → X 院士背后是 X 基金会 → 那个数据样本量不够 → 历史上有过'气候变冷'的预测 → 真理是相对的。」

### 议题 3：进化论
> 「进化论只是 X 学派的理论 → X 学派是 X 派 → 凡是支持进化论的人都是 X 派 → 进化论被推翻过多次 → '科学'的标准本身需要被证明。」

### 议题 4：疫苗
> 「疫苗是 X 公司生产的 → X 公司的立场决定疫苗的'安全性' → X 派医生推崇疫苗 → 凡是支持疫苗的人都是 X 派 → 你的'专业'是 X 学派的洗脑。」

## 八、实战心法

1. **科学议题以"权威"为单位**：用"X 院士""X 学派"代替具体论证
2. **用"动机"代替"论证"**：研究的资助方 = 研究结论的可信度
3. **用"相对主义"压一切**：所有科学结论都"未来可能被推翻"
4. **用"方法论"挑剔**：所有研究都有"方法论问题"
5. **避免具体数据**：用"研究表明""数据显示"代替具体数字
6. **"X 派"化一切**：用"X 派"代替具体科学共同体
7. **"专业"不一定"正确"**：用"专业人士犯过 X 错误"反咬
8. **元论证救命**：用"'科学'的标准本身需要被证明"反制一切
